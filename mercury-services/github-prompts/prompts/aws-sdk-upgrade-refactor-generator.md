## Placeholder reference

| Placeholder | Source | Notes |
|-------------|--------|-------|
| `{{MODULE}}` | `module` (mandatory) | module name |
| `{{DOC}}` | `doc` (mandatory) | output documentation file path |
| `{{GOALS_BULLETS}}` | `goals` (mandatory) | the goals rendered as a bullet list |
| `{{ONE_LINE_GOALS_SUMMARY}}` | derived from `goals` | short single-line summary for descriptions/commit |
| `{{BRANCH}}` | `branch` (mandatory) | feature branch; created from latest develop if absent, else rebased onto develop |
| `{{REFERENCES_BULLETS}}` | `references` (optional) | one or more reference files as bullets; section dropped if blank |
| `{{JIRA}}` | `jira` (mandatory) | Jira key |
| `{{CONFLUENCE}}` | `confluence` (optional) | Confluence URL; `{{#CONFLUENCE}}…{{/CONFLUENCE}}` dropped if blank |
| `{{WORK_ID}}` | = `jira` | token required in the commit message + session tags |
| `{{YYYYMMDD}}` | today (UTC) | used in filename, backups, session name |
| `{{BRIEF_SUMMARY}}` | derived from `goals` | kebab-case slug for the filename |

The `{{#X}}…{{/X}}` blocks are conditional: include them only when input `X` is provided. Only `references` and
`confluence` are optional. Never leave a raw `{{...}}` token or an empty heading in the generated file.

The `{{#X}}…{{/X}}` blocks are conditional: include them only when input `X` is provided (for `#NO_X`, include only
when `X` is **absent**). Never leave a raw `{{...}}` token or an empty heading in the generated file.
