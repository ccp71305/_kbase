# Anthropic Tutorials — Complete Learning Plan

A sequenced learning plan (beginner → advanced) covering Anthropic's official tutorials, with direct links to the real resources.

There are three "homes" for Anthropic's tutorials:
- **Anthropic Academy** (skilljar) — polished, self-paced video courses with completion certificates
- **GitHub `anthropics/courses`** — hands-on Jupyter-notebook courses (use Claude 3 Haiku to keep costs low)
- **Docs + Cookbook + Quickstarts** — reference tutorials and runnable example apps

---

## Phase 1 — Foundations (Prompting & API basics)

- [ ] 1. **Anthropic API Fundamentals** (API key, params, multimodal, streaming) — GitHub — https://github.com/anthropics/courses/tree/master/anthropic_api_fundamentals
- [ ] 2. **Prompt Engineering Interactive Tutorial** (9 chapters + appendix) — GitHub — https://github.com/anthropics/courses/blob/master/prompt_engineering_interactive_tutorial/README.md
- [ ] 3. **Intro to Claude / Quickstart** (first API call) — Docs — https://platform.claude.com/docs/en/intro
- [ ] 4. **Claude with the Anthropic API** (Academy version, certificate) — Academy — https://anthropic.skilljar.com/claude-with-the-anthropic-api

---

## Suggested pacing

- **Week 1–2:** Phase 1–2 (prompting + API foundations)
- **Week 3:** Phase 3 (tools & agents)
- **Week 4:** Phase 4–5 (MCP + Claude Code) — most relevant to day-to-day work
- **As needed:** Phase 6–7

The GitHub `courses` repo recommends this internal order:
**API Fundamentals → Prompt Engineering → Real World Prompting → Prompt Evaluations → Tool Use**, which Phases 1–3 above follow.
